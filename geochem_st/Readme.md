# Geochemistry Streamlit Dashboard

A comprehensive web application for geochemical data analysis built with Streamlit.

## Features

- **Data Overview**: Interactive exploration of geochemical datasets
- **Element Distribution**: Statistical analysis and visualization of element concentrations
- **Correlation Analysis**: Multivariate analysis including PCA and clustering
- **Spatial Analysis**: Geographic visualization and mapping of sample locations
